# 調査結果 → 契約書（重説・売買契約書）反映マッピング

> このファイルは、不動産調査で得られた情報がどの契約書のどの箇所に反映されるべきかを網羅的にまとめたもの。
> 調査結果を受け取ったら、このマッピングに従って重説・契約書に漏れなく記入する。

## 6. 特記事項として契約書に入れ込む必要がある調査結果

### ★最重要：「III その他重要な事項」への自動反映

以下の調査結果が該当する場合、重説の「III その他重要な事項」セクションに特記事項として記載が**必須**。

| 調査結果 | 特記事項の記載内容（テンプレート） |
|---------|---------------------------|
| **埋蔵文化財包蔵地に該当** | 「◆本物件は文化財保護法に定める周知の埋蔵文化財包蔵地内にあるため、建築工事を行う場合には事業着手の60日前までに教育委員会への届出が必要となります。届出の結果、試掘が必要となる場合（調査期間および調査費用が生じます）や事業中止、事業の変更の指示を受ける場合があります。」 |
| **宅地造成等規制区域内** | 「◆本物件は宅地造成等規制区域内にあるため、次の行為を行う場合には都道府県知事の許可が必要となります。（以下、具体的行為を列挙）」 |
| **土壌汚染（要措置区域等）** | 「◆本物件は土壌汚染対策法に基づく〇〇区域に指定されています。（区域の種類と対応措置を記載）」 |
| **アスベスト使用の可能性** | 「◆現在すでに建築されている多くの建物には、石綿（アスベスト）を含む建材が使用されている可能性があります...」（重説の13.石綿使用調査セクションにも記載） |
| **洪水浸水想定区域内** | 12.水害ハザードマップセクションに記載 + 特記事項にも詳細を記載 |
| **固定資産税等の精算** | 「◆上記固定資産税等清算金の起算日はX月X日です。」 |
| **自治会加入の推奨** | 「◆当該物件区域内に自治会がある場合には規約等を遵守するものとし、地域活動等の協力をお願い致します。」 |
| **法令に基づく制限の将来変更** | 「◆「法令に基づく制限」については、重要事項説明時点における内容であり、将来法令の改正等により、本物件の利用に関する制限が付加または緩和されることがあります。」 |
| **契約書の作成通数・印紙** | 「◆契約書は1通作成、印紙代は買主負担とし、原本を買主、その写しを売主が所有させて頂きます。また、契約当日に必ず印紙を添付し、買主の割り印を頂きます。」 |

## 8. ユーザーに「取りに行かせる」判断基準

以下の資料が不足している場合、ユーザーに取得を依頼する:

### 必須資料（なければ契約書作成不可）
- 登記事項証明書（土地・建物）
- 公図
- 用途地域・建蔽率・容積率の情報
- 前面道路の種別と幅員
- 上下水道の埋設管情報

### あれば反映する資料（なくても作成可だが品質が落ちる）
- 地積測量図
- 建物図面
- 建築台帳記載事項証明書
- ハザードマップ
- 公課証明書（固定資産税精算の根拠）
- 埋蔵文化財包蔵地の確認結果
- おおいたマップの印刷

### 物件タイプ別の追加必須資料
- **マンション**: 管理規約、修繕積立金額、管理組合議事録
- **農地**: 農地転用許可書または届出受理書
- **借地権付**: 借地契約書の写し
