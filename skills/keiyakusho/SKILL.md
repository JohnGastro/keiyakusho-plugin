# ラビーネット契約書スキル

## 概要

全日本不動産協会（ラビーネット）の統一書式に準拠した不動産契約書をCowork上で作成・記入するスキル。売買、賃貸借、媒介契約、覚書など、複数の契約形態と物件タイプに対応。

## ワークフロー

### ステップ1: 取引形態の特定

利用者に以下を確認する:

- **取引形態**: 売買 / 賃貸借 / 媒介契約 / 覚書 / その他
- **物件タイプ**: 土地 / 土地建物 / 区分所有建物（マンション）/ 借地権付建物 / 駐車場 / 事業用建物
- **契約形態** (売買): 一般仲介用 / 消費者契約用 / 売主宅建業者用
- **保証形態** (賃貸借): 連帯保証人型 / 家賃債務保証業者型 / 併用型
- **借家形態** (賃貸借): 普通賃貸借 / 定期賃貸借

### ステップ2: テンプレート選定

`references/書式インデックス.md` を参照して最適なテンプレートを特定する。通常は重要事項説明書、契約書、付帯書類を一括で選択する。

テンプレートの選択にはフローチャートを使用する:

```
売買
├─ 一般仲介 → 物件タイプ別に 11-1 ～ 19-1
├─ 消費者契約 → 物件タイプ別に 21-1 ～ 29-1
└─ 売主宅建業者 → 物件タイプ別に 31-1 ～ 41-1

賃貸借
├─ 居住用 → 普通/定期 × 保証形態別に 51-1 ～ 52-3
├─ 土地 → 56-1 ～ 57-3
├─ 事業用 → 61-1 ～ 64-1
└─ 駐車場 → 54-1

媒介契約
├─ 売買媒介 → 一般201 / 専任202 / 専属専任203
└─ 賃貸借媒介 → 501 ～ 505
```

### ステップ3: 調査データの受け取りと不足チェック

ユーザーが物件の調査データ（PDFファイル、テキスト情報、物件フォルダ等）を渡してきた場合、`references/調査結果_契約書マッピング.md` を参照して以下を行う:

1. **受け取ったデータの確認**: どの調査資料が揃っているか棚卸しする
2. **不足資料の特定**: マッピングの「必須資料」リストと照合し、不足があればユーザーに取得を依頼する
3. **特記事項の自動検出**: 以下の調査結果は重説の「III その他重要な事項」に入れ込む必要がある:
   - **埋蔵文化財包蔵地に該当** → 60日前届出義務、調査費用・工期遅延の可能性を記載
   - **宅地造成等規制区域内** → 許可が必要な行為を列挙
   - **土壌汚染（要措置区域等）** → 区域の種類と対応措置を記載
   - **洪水浸水想定区域内** → 水害ハザードマップセクション + 特記事項に詳細記載
   - **土砂災害警戒区域内** → 該当区域を記載
   - **アスベスト使用の可能性** → 石綿使用調査セクションにも記載
4. **添付書類リストの自動生成**: 取得した資料を「IV 添付書類」に番号付きで記載する

調査データがない場合は、従来どおりユーザーからの口頭ヒアリングで情報を収集する。

### ステップ4: 必要情報のヒアリング（不足分）

調査データで埋まらない情報を利用者に確認する: 当事者情報（売主/買主/賃貸人/賃借人の氏名・住所・電話番号）、取引条件（価格/賃料、手付金、支払期日、引渡日、契約日）、特約事項（ローン特約、契約不適合責任の範囲）。

### ステップ5: テンプレートへの流し込み

`scripts/fill_template.py` でExcelテンプレートに自動入力する。`scripts/utils.py` の以下ユーティリティを使用:

- `DateConverter.to_japanese_format()` — 和暦変換（令和X年X月X日）
- `CurrencyFormatter.to_format()` — 金額フォーマット
- `CurrencyFormatter.to_man()` — 万円表示
- `ContractCalculator.calculate_brokerage_fee()` — 仲介手数料計算（dictで返る: base, tax, total, description）
- `ContractCalculator.calculate_stamp_duty()` — 印紙税計算
- `ContractCalculator.calculate_remaining_price()` — 残代金計算

### ステップ6: 検証とお渡し

以下を検証する:
- 記入漏れ（特に調査結果から反映すべき特記事項の漏れ）
- 金額の整合性（売買代金 = 手付金 + 残代金）
- 添付書類リストと実際の添付書類の一致
- 埋蔵文化財・ハザードマップ等、調査結果由来の特記事項が「III その他重要な事項」に反映されているか

完成したExcelファイルを物件フォルダに保存して渡す。

## 重要な注意点

### 2020年民法改正

- **契約不適合責任**（旧:瑕疵担保責任）の範囲明記が必要
- 賃貸借で連帯保証人を立てる場合、**極度額の明記**が推奨
- 定期賃貸借は事前説明・合意が必須、説明書交付後2週間以上の検討期間が必要

詳細は `references/法改正対応.md` を参照。

### テンプレートの利用規約

ラビーネットのテンプレートは全日本不動産協会会員団体のみの利用を想定。自社内での利用が前提。
