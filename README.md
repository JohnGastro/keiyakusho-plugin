# keiyakusho プラグイン

全日本不動産協会（ラビーネット）の統一書式に準拠した不動産契約書の作成と、不動産調査ワークフローを支援するCoworkプラグインです。

## 機能

### スキル

- **keiyakusho** — ラビーネット書式での契約書作成（売買、賃貸借、媒介契約、覚書、付帯書類）
- **fudosan-chosa** — 不動産調査ガイド、チェックリスト生成、物件フォルダ管理（大分市特化の窓口案内付き）

### コマンド

| コマンド | 説明 |
|---------|------|
| `/shinki-bukken` | 新しい物件フォルダを作成 |
| `/chosa-checklist` | 不動産調査チェックリストを生成 |
| `/keiyakusho-sakusei` | ラビーネット書式で契約書を作成 |

## セットアップ

### 前提条件

- ラビーネットのExcelテンプレートファイル（`templates/` フォルダ）が必要です
- テンプレートはラビーネット会員サイトからダウンロードしてください

### テンプレートの配置

プラグインと同じフォルダ、またはワークスペース内の `templates/` に以下のカテゴリでテンプレートを配置してください:

```
templates/
├── 売買_重説契約書/
├── 売買_媒介契約/
├── 売買_関係書式/
├── 売買_付帯書式/
├── 売買_覚書/
├── 賃貸_重説契約書/
├── 賃貸_媒介契約/
├── 賃貸_関係書式/
├── 賃貸_条例対応/
├── 業務関係/
├── 補足資料/
└── マスターリース/
```

### Python依存パッケージ

契約書自動入力機能を使う場合は `openpyxl` が必要です:

```bash
pip install openpyxl
```

## 使い方

### 契約書の作成

「マンションの売買契約書を作りたい」「賃貸借契約書を作成して」などと伝えると、スキルが起動してテンプレート選定から記入までガイドします。

### 物件フォルダの作成

「荷揚町の物件フォルダを作って」と伝えるか、`/shinki-bukken 荷揚町マンション 大分市荷揚町2-31` のようにコマンドを使います。

### 調査チェックリスト

「土地建物の調査チェックリストを作って」と伝えるか、`/chosa-checklist 土地建物` を使います。大分市の窓口情報（何課・何階）付きのチェックリストが生成されます。

## 対応エリア

現在は **大分市** に特化した窓口情報を提供しています。他の市区町村については調査マニュアルの一般的な手順を参照してください。

## ライセンス

ラビーネットのテンプレートは全日本不動産協会会員のみ利用可能です。テンプレートファイル自体はこのプラグインには含まれません。
