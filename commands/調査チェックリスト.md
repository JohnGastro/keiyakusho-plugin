不動産調査チェックリストを生成する。

1. 引数から物件タイプを取得する。引数がない場合はAskUserQuestionで聞く:
   - 物件タイプ: 土地 / 土地建物 / マンション（区分所有） / 借地権付建物 / 事業用 / 駐車場 / 農地

2. fudosan-chosaスキルの以下を読み込む:
   - `references/不動産調査マニュアル.md` （全体の調査手順）
   - `references/大分市_調査ガイド.md` （大分市の窓口・Web情報源）

3. 物件タイプに応じたチェックリストをマークダウン形式で生成する。以下のセクションを含める:

   **事前準備（Web調査）**
   - おおいたマップでの確認項目
   - 登記情報の確認項目
   - ストリートビューでの確認項目

   **法務局調査**
   - 取得すべき資料一覧（公図、登記事項証明書、地積測量図等）

   **現地調査**
   - 境界標、道路幅員、インフラ、周辺環境の確認項目

   **役所調査**（大分市の窓口情報付き）
   - 各課での確認事項と窓口の場所（階数含む）

   **インフラ調査**
   - 上水道、下水道、ガスの確認項目

4. 各項目にはチェックボックス `- [ ]` を付け、窓口情報（場所・電話番号）も併記する。

5. 物件タイプ別の追加項目（マンション→管理組合、農地→農業委員会等）も含める。

6. 完成したチェックリストをファイルとして保存し、リンクを返す。
