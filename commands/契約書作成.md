ラビーネット（全日本不動産協会）の統一書式で不動産契約書を作成する。

1. keiyakushoスキルを読み込む（SKILL.mdとreferences/書式インデックス.md）。

2. 引数から契約種類を取得する。引数がない場合はAskUserQuestionで以下を順に確認する:
   - 取引形態: 売買 / 賃貸借 / 媒介契約 / 覚書
   - 物件タイプ: 土地 / 土地建物 / マンション / 借地権付建物 / 駐車場 / 事業用
   - 契約形態の詳細（売買なら一般仲介/消費者契約/売主宅建業者、賃貸借なら保証形態・借家形態）

3. `references/書式インデックス.md` を参照して、適切なテンプレートファイルを特定する。

4. 利用者から必要情報を収集する:
   - 物件情報（所在地、面積、構造、築年数、用途地域）
   - 当事者情報（氏名、住所、電話番号）
   - 取引条件（価格/賃料、手付金、支払期日、引渡日、契約日）
   - 特約事項

5. `scripts/fill_template.py` と `scripts/utils.py` を使ってExcelテンプレートに自動入力する。

6. 記入漏れや金額の整合性を確認する。

7. 完成したファイルを物件フォルダ（あれば）の `06_契約書類/` に保存するか、ワークスペースに保存してリンクを返す。
